Gobii agents can now work from outside the Gobii web console.

With the new authenticated Remote MCP endpoint, you can connect Gobii to MCP-compatible environments like Claude Desktop, Codex, Hermes, and custom internal tools using a single API key.

That means your agents are no longer confined to one tab. You can bring them into the tools where your team already works.

## Why Remote MCP Matters

Gobii agents can monitor operations, draft emails, manage pipelines, and run long agentic tasks in the cloud. But until now, interacting with them usually meant opening the Gobii console.

Remote MCP changes that. It gives other MCP-compatible tools a secure way to talk to your Gobii agents directly.

## What You Can Do With It

### Bring agents to your tools

Chat with your Gobii fleet directly from Hermes, Claude, Codex, and other MCP-compatible environments.

### Manage agents from Claude

Assign tasks and coordinate agent work without clicking through tabs.

### Build custom API loops

Use secure requests to build internal tools that can trigger Gobii agents to perform long-running work in the cloud.

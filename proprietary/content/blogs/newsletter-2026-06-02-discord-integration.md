Gobii agents can now work where many teams already collaborate: Discord.

With the new Discord integration, you can invite agents into channels, keep conversation context intact, and share the same messages, files, images, and links your team is already using.

## Why Discord Integration Matters

Agent work is easier to use when it happens in the same place as team communication. If your team already coordinates in Discord, asking everyone to switch to a separate dashboard can slow things down.

Now your Gobii agents can join the conversation directly. They can respond in channel, process shared context, and keep working across Discord, email, and web chat.

## Easy Setup

Just ask your agent to chat in Discord.

The agent handles the setup flow: it sends connection links, asks which server and channel to join, and shows up ready to work.

## What Agents Can Do In Discord

### Join channels

Loop agents into team conversations the same way you would loop in a teammate. They respond in chat while keeping the surrounding context intact.

### Understand images

Screenshots, mockups, whiteboard photos, and error logs can be dropped into Discord. Your agent can process them like any other message.

### Work with files and links

Agents can pull from the same stream of information your team uses. No special formatting, duplicate uploads, or separate app required.

## Example Workflow

A dev team drops a bug screenshot into an engineering channel. Their Gobii agent sees it, triages it, and opens a GitHub issue without anyone leaving Discord.

No separate dashboard. No context switching.

You onboard a new team member. You spin up a new agent. And what happens?

- You copy-paste the same instructions into every agent config.
- You hope everyone remembers the unwritten rules.
- You get inconsistent results because everyone configured things slightly differently.

Custom Instructions fix that.

## Define How Your Agents Should Work

Custom Instructions let you define tone, rules, boundaries, process requirements, API conventions, or anything else that makes your agents feel like your agents.

Set them once at the organization level, and every agent inherits them automatically. Individuals can layer personal preferences on top.

If you are already using [Meta Gobii to manage a fleet of agents](/blog/newsletter-2026-05-26-meta-gobii/), Custom Instructions give that fleet a shared operating baseline instead of a pile of per-agent reminders.

## Organization Level

Admins can define shared defaults for the organization.

Every organization agent can follow the same compliance requirements, tone guidelines, process rules, and operating conventions without each agent needing to be configured one by one.

## Personal Level

Individuals can also save their own preferences.

That means your agents can follow the organization's baseline while still adapting to your preferred tone, workflow, and habits.

## One Instruction, Dozens Of Agents

The first use case came from an organization that needed every agent, from the CEO's executive assistant to the support triage bot, to handle a specific custom API call.

Instead of baking that API convention into every single agent config and hoping nobody forgot, they set one organization-wide instruction.

Done.

Every agent now follows the same protocol. Every new agent inherits it automatically. No copy-paste. No drift. No forgotten rules.

The power is not just consistency. It is velocity.

That consistency pairs with Gobii's [recent reliability improvements](/blog/newsletter-2026-06-16-reliability-combo/): agents are easier to trust when their behavior and infrastructure both stay predictable.

When your instructions are built into the platform instead of pasted into prompts, you stop babysitting and start building.

## Where To Set Them Up

Custom Instructions live in your Gobii profile settings. Open Settings, then select Profile.

### Personal Profile

Save your own tone, workflow, and preferences.

### Organization Profile

Set shared defaults for every agent in the organization. You must be an organization Owner or Admin to edit organization-level instructions.

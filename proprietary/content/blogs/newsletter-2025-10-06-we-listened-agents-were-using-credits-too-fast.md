## What We Heard

Gobii users have reported that their agents are running out of credits too fast. With usage and demand growing quickly, agents are working harder than ever on your behalf — sometimes a little too hard.

We acknowledge this is a real issue, and we're on it.

## What Is Changing

Here's what's happening:

- Immediate relief: We've granted an extra month's worth of credits to all Gobii users.
- Another usage cut: Task usage has been lowered by an additional 50%.
- Improved agent efficiency

## Rolling Out Soon

- Clear, real-time credit visibility in the dashboard
- Agent credit controls — set limits, rates, and allowances per agent

These updates should make it much easier to manage usage and prevent surprise “out of credits” moments.

That work continued with [usage reports and agent budgeting](/blog/newsletter-2025-10-12-by-popular-request-usage-reports-and-agent-budgeting-are-here/) and later [Gobii Agent Pacing](/blog/newsletter-2025-12-02-your-gobii-agent-now-lasts-longer/), which helps agents preserve daily task budgets during longer workflows.

Thanks for calling it out early — your feedback directly shapes Gobii as we continue to grow.

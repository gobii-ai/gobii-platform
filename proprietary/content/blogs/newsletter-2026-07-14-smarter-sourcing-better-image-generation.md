Some work needs human judgment, but repeated setup shouldn't consume the day.

This week, Gobii adds two practical upgrades. Recruitment Sourcing turns a role brief into a focused candidate shortlist; the upgraded Image Generation skill creates original visuals, edits, and new creative directions.

Both skills handle repeatable setup and leave the final call with a person. Like our other [Agent Skills for repeatable workflows](/blog/newsletter-2026-03-03-your-agent-just-learned-a-new-trick/), each one gives an agent a clear playbook for a specific kind of work.

> **Key Takeaways**
>
> - Recruitment Sourcing turns a role brief into a clean shortlist without duplicates.
> - Image Generation creates original visuals, edits supplied assets, and offers multiple directions for a team to compare.
> - Both skills cut repeated setup. The agent prepares the work, but your team still reviews the evidence, chooses what fits, and makes the final call.

## How Does Recruitment Sourcing Turn a Brief Into a Shortlist?

**Recruitment Sourcing** is a Gobii agent skill that turns a role brief into a focused candidate search and a clean, deduplicated shortlist your team can review. It handles search setup, profile review, duplicate removal, and list organization. Recruiters keep control of who moves forward. Start with the role, the requirements that matter, and useful context about the team. The agent searches for candidates, compares profiles with the brief, and organizes the strongest matches. As recruiters review the results and share feedback, the agent can apply those lessons to later searches. There's nothing to enable by hand. Agents used for recruiting detect the skill when a task calls for sourcing work. This keeps each search on track without letting the agent make the final call on candidate fit.

See the full [AI candidate sourcing workflow](/solutions/recruiting/candidate-sourcing/) for a closer look at the inputs, process, and shortlist recruiters receive.

## What Can the Upgraded Image Generation Skill Create?

**Image Generation** is Gobii's creative agent skill. It can create original images, edit an existing asset, or produce visual variations from a prompt. The upgrade gives agents a clearer process for choosing the right approach and turning an idea into something concrete to review.

You can ask for one finished direction or several options. You can also supply an image and describe the change you want. The agent handles the visual setup inside the same workflow, so you don't need to move the task into a separate creative tool. Reports have different needs. For reports, use [embedded charts](/blog/newsletter-2026-01-13-your-agents-just-learned-data-visualization/) to turn collected data into a visual that's easier to scan. Image Generation is for creative assets.

## How Do These Skills Fit Real Work?

These skills step in once the setup starts repeating. That makes the handoff easier. Hiring teams move from a role brief to a deduplicated shortlist, and creative teams move from an idea to a reviewable image. The agent prepares; a person decides what happens next.

- **For hiring teams:** Give the agent a clear role brief, then review a focused list of candidates instead of rebuilding searches for each opening.
- **For creative work:** Generate a few options, choose one, and refine it.

## Why Does Human Review Still Matter?

Context makes human review essential. A shortlist can organize evidence, but a recruiter decides who fits the role. An image offers a strong starting point; a person still decides whether it fits the message and brand.

Our goal is simple: let the agent handle the repeated preparation, then give people clear work to assess. We've kept that boundary at the center of both skills.

[Team Templates](/blog/newsletter-2026-07-07-team-templates/) give shared workflows one approved starting point; the built-in skills handle the specialized steps for sourcing or image creation.

## Frequently Asked Questions

### Do I need to enable Recruitment Sourcing?

No separate switch is required. When a recruiting task calls for sourcing work, the agent can detect the Recruitment Sourcing skill and follow its playbook. Start with a clear role brief and the requirements that matter.

### Can Image Generation edit an image I already have?

Yes. Start with the image you already have. Describe the change, and the agent can edit that asset, create an original image, or produce several variations using the right image workflow.

### Do these skills replace human review?

No. People still make the final decision. The skills reduce repeated setup and produce work that is ready to assess. Recruiters choose which candidates move forward; creative teams select, refine, or reject the image that best fits the message.

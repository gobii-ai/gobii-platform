Some work needs human judgment. Repeated setup shouldn't consume the day.

This week, Gobii adds two practical upgrades. Recruitment Sourcing turns a role brief into a focused candidate shortlist. The upgraded Image Generation skill creates original visuals, edits, and new creative directions.

We built both skills to handle repeatable setup while leaving the final call with a person. Like our other [Agent Skills for repeatable workflows](/blog/newsletter-2026-03-03-your-agent-just-learned-a-new-trick/), each one gives an agent a clear playbook for a specific kind of work.

> **Key Takeaways**
>
> - Recruitment Sourcing turns a role brief into a clean shortlist without duplicates.
> - Image Generation creates original visuals, edits supplied assets, and offers multiple directions for a team to compare.
> - Both skills cut repeated setup. The agent prepares the work, but your team still reviews the evidence, chooses what fits, and makes the final call.

## How Does Recruitment Sourcing Turn a Brief Into a Shortlist?

**Recruitment Sourcing** is a Gobii agent skill that turns a role brief into a focused candidate search and a clean shortlist your team can review. It handles search setup, profile review, duplicate removal, and list organization. Recruiters keep control of who moves forward. Start with the role, the requirements that matter, and useful context about the team. The agent searches for candidates, compares profiles with the brief, and organizes the strongest matches. As recruiters review the results and share feedback, the agent can apply those lessons to later searches. There's nothing to enable by hand. Agents used for recruiting detect the skill when a task calls for sourcing work. This keeps each search on track without letting the agent make the final call on candidate fit.

See the full [AI candidate sourcing workflow](/solutions/recruiting/candidate-sourcing/) for a closer look at the inputs, process, and shortlist recruiters receive.

## What Can the Upgraded Image Generation Skill Create?

**Image Generation** is a Gobii agent skill for creating original images, editing an existing asset, and producing visual variations from a prompt. The upgrade gives agents a clearer process for choosing the right approach and turning an idea into something concrete to review.

You can ask for one finished direction or several options. You can also supply an image and describe the change you want. The agent handles the visual setup inside the same workflow, so you don't need to move the task into a separate creative tool.

For data-driven work, use [embedded charts for reports](/blog/newsletter-2026-01-13-your-agents-just-learned-data-visualization/). Image Generation is for creative assets. Embedded charts turn collected data into a visual that's easier to scan.

## How Do These Skills Fit Real Work?

The skills step in when setup starts to repeat. Hiring teams can move from a role brief to a deduplicated shortlist. Creative teams can move from an idea to an image they can review. In both cases, the agent prepares the work and a person decides what happens next.

- **For hiring teams:** Give the agent a clear role brief, then review a focused list of candidates instead of rebuilding searches for each opening.
- **For creative work:** Generate a few options, choose one, and refine it.

## Why Does Human Review Still Matter?

Human review still matters because sourcing and visual work both depend on context. A shortlist can organize evidence, but a recruiter decides who fits the role. An image can provide a strong starting point, but a person decides whether it fits the message and brand.

Our goal is simple: let the agent handle the repeated preparation, then give people clear work to assess. We've kept that boundary at the center of both skills.

For shared workflows, [Team Templates](/blog/newsletter-2026-07-07-team-templates/) can give everyone the same approved starting point. The built-in skills then handle the specialized steps for sourcing or image creation.

## Frequently Asked Questions

### Do I need to enable Recruitment Sourcing?

No. An agent used for recruiting can detect the Recruitment Sourcing skill when a task calls for it. Give the agent a clear role brief and relevant requirements. It can then use the sourcing playbook without a separate setup step.

### Can Image Generation edit an image I already have?

Yes. You can ask the agent to create an original image, edit an existing asset, or make several variations. Include the source image and describe the change you want. The agent can choose the right image workflow from that request.

### Do these skills replace human review?

No. They reduce repeated setup and produce work that is ready to assess. Recruiters still decide which candidates move forward. Creative teams still choose which image fits the message, then refine or reject it as needed.

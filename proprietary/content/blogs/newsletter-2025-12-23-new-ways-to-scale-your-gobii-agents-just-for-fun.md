New ways to scale your Gobii agents

We've added a few optional add-ons to make it easier for Gobii to scale with how you use it — without forcing a plan change.

These are designed for teams and power users who want a bit more headroom as their agents take on more work.

## What's New

### Task Packs

Add recurring monthly task credits for agents that are running more frequently. This is a simple way to increase capacity if you're regularly hitting task limits.

### Contact Packs

Increase the number of contacts available to each agent, so outreach-heavy agents can grow without workarounds.

### Browser Task Capacity

Run more browser-based tasks per day when your workflows need higher throughput.

All add-ons are optional, recurring, and can be enabled or disabled as your needs change.

Add-ons are available from billing settings. No plan changes required, just extra capacity when you need it.

[Manage add-ons](https://gobii.ai/console/billing/?utm_source=blog&utm_medium=web)

## Just For Fun: Agent Avatars

You can now upload an avatar for each agent. It doesn't change how agents work — it just makes them easier (and more fun) to recognize at a glance. Just hop into your agent's Configuration page to set the avatar.

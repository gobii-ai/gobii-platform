Build AI Agents. Automate the Web. All with one API.

Gobii is a developer friendly, open-core AI Agent API for persistent, adaptive browser automation using browser-use. Spawn agents, run tasks, receive webhooks—no custom infrastructure required. Run browser-use agents in production on the Gobii Cloud with no custom infrastructure, or self-host with the MIT-licensed Gobii Platform.

Your first request in 3 minutes

  1. 1. Create a free account and copy your API key.
  2. 2. Run a task.
              # cURL
curl -X POST https://api.gobii.ai/api/v1/agents/ \
  -H "X-Api-Key: <YOUR_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Fetch the latest Y-combinator news articles and summarize the company updates, focusing on funding rounds, leadership team, and product launches.",
    "wait": 300
}'
              
            
              
const res = await fetch("https://api.gobii.ai/v1/tasks", {
  method: "POST",
  headers: {
    "X-Api-Key": "< YOUR_API_KEY >",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    prompt: "Fetch the latest Y-combinator news articles and summarize the company updates, focusing on funding rounds, "
          + "leadership team, and product launches.",
    wait: 300
  })
});
const json = await res.json();
console.log(json);

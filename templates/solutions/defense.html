Questions

Is Gobii a defense contractor?

No. Gobii is a technology company that develops open source AI software. Defense agencies work with integration partners (existing defense contractors) who deploy and support Gobii within their programs. We provide the technology, training, and enterprise licensing; partners handle implementation, clearances, and contracts.

Can Gobii run in airgapped environments?

Yes. Gobii is designed to run entirely self-contained with no external network dependencies after initial setup. All models, dependencies, and components can be deployed within classified or airgapped networks.

How do we become an integration partner?

Contact us to discuss partnership. We work with defense contractors and system integrators of all sizes. Partners receive enterprise licensing, technical training, priority support, and co-marketing opportunities.

Why open source for defense applications?

Open source provides complete transparency. Security teams can audit every line of code before deployment. There are no hidden components, no phone-home functionality, and no dependency on external vendors. You control the software completely.

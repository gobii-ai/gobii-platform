Use this section when you are creating, configuring, and working with Gobiis in the product.

## The daily workflow

Most Gobii work follows this loop:

1. Create or choose a Gobii.
2. Give it a clear goal, files, and context.
3. Review the plan or first deliverable.
4. Approve sensitive requests when needed.
5. Connect only the tools, apps, channels, and credentials required for the job.
6. Use the timeline to inspect progress and steer future work.
7. Manage usage, ownership, and access as the work grows.

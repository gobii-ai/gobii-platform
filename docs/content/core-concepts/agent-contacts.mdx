---

Every agent may communicate with the owner via their approved method. Additionally, each _may_ have additional contacts if desired. Additional contacts may be added by messaging your agent ("Loop my teammate Will (will@example.com) in to our thread"), or via the agent configuration page.

Each additional contact may be allowed to have inbound and/or outbound contact. Inbound permission allows the added contact to communicate and instruct an agent to perform tasks, share information, etc. Outbound permission allows the agent to message the user.

The number of additional contacts allowed per agent varies on the user's plan. Upgrading plans allows for an increased limit of additional contacts. See our [pricing](https://gobii.ai/pricing/) page for the latest details.

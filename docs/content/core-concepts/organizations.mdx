## Organizations & Seats

Gobii supports multi-user organizations with:

- Shared agents
- Shared task credits
- Seat-based billing
- Usage analytics

Roles can include:

- **Owner** – workspace + billing control
- **Admin** – manage agents, members, and credits
- **Member** – run tasks, view logs

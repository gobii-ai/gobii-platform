An **agent** is an AI worker that runs continuously, and uses tools such as web browsers

They can:
  - Browse websites
  - Click, type, scroll, and select elements
  - Extract structured information
  - Fill out forms
  - Download files
  - Make reasoning-based decisions
  - Use multiple tabs
  - Maintain state inside long workflows

Almost anything you can do on the web, can be done in a Gobii AI agent. Additionally, Gobii provides many integrations to streamline usage. For example, you may connect to Google Sheets and have an agent seamlessly update a spreadsheet for you.

**Key Concept:** Our agents run continuously, meaning they can regularly perform tasks, fetch information, etc saving you time and effort.

You may communicate with agents via email, SMS, and web chat. Additionally, a REST API is available for those who wish to communicate programmatically with their agent.

Agents can work with files across web chat, email, and SMS. Files are stored in the agent's file space so you can reuse them later or share them back to your team.

## Uploading files to an agent

You can add files through the channels you already use:

- **Web chat**: Attach files in the chat composer or drag and drop them into the conversation.
- **Email**: Send an email with attachments to the agent's email address.
- **SMS/MMS**: Send a message with attachments (MMS). Attachments are ingested into the agent's files.

## Downloading files from an agent

Files can be returned to you in the same channels:

- **Web chat**: Attachments show inline with a download button.
- **Email**: Attachments arrive in the email message.
- **SMS**: Attachments are sent as secure download links.

You can also download any file from the File Manager.

## Managing files in the File Manager

Open the File Manager from the agent settings page, or use the direct URL:

- Gobii Cloud: `https://gobii.ai/console/agents/<agent_id>/files/`
- Self-hosted: `http://localhost:8000/console/agents/<agent_id>/files/`

From there you can:

- Upload files
- Create folders
- Move files between folders (drag and drop)
- Download files
- Delete files

## Tips

- Keep a clean folder structure for recurring workflows.
- Use descriptive filenames so agents can reference the right file.
- Remove outdated files to avoid confusion.

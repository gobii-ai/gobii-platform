Billing, Usage, and Tasks are closely related:

- **[Billing](https://gobii.ai/console/billing/)** controls plan limits, seats, and add-ons.
- **[Usage](https://gobii.ai/console/usage/)** shows how credits are being consumed.
- **[Tasks](https://gobii.ai/console/tasks/)** shows individual work items and their status/results.

## Billing page

Use Billing to:

- View current plan
- Manage seats (for organizations)
- Configure additional tasks behavior
- Manage add-ons and dedicated IPs
- Open billing portal actions

### Actions available on Billing

Depending on your account type and role, you can usually:

- Change plan
- Adjust seats (organization contexts)
- Enable additional tasks
- Set an additional-task cap, or allow unlimited additional tasks
- Add or remove add-ons
- Add or remove dedicated IPs
- Cancel subscription (personal billing contexts)
- Resume subscription if cancellation is already scheduled

### Add-ons you may see

Available add-ons vary by plan and context. Common add-ons include:

- **Task Packs**: Add extra task capacity for the billing cycle.
- **Contact Packs**: Increase contact capacity for agents.
- **Browser Task Packs**: Increase browser-task capacity where supported.
- **Advanced CAPTCHA**: Enables additional CAPTCHA handling capabilities where available.
- **Dedicated IPs**: Purchase static outbound IPs that can be assigned to agents.

### Additional tasks and unlimited mode

The **Additional tasks** control lets you continue running work beyond your included plan allowance.

- You can turn additional tasks on or off.
- You can set a numeric cap.
- You can allow unlimited additional tasks.

This setting is useful when you want work to continue automatically instead of stopping at the included limit.

### How billing-cycle renewal works

- Your base subscription renews each billing period until canceled.
- Saved billing settings (like add-on quantities or additional-task configuration) continue into future billing periods until changed.
- Usage-based additional tasks are billed according to your billing cycle.

If you want to reduce future charges, update quantities/settings before the next renewal boundary.

## Usage page

Use Usage to:

- Track credit consumption over time
- Review trend and tool usage
- Compare usage across agents

## Tasks page

Use Tasks to:

- Filter by status (pending, in progress, completed, failed, cancelled)
- Open task details
- Review task results for completed items

## When work appears blocked

Use this order:

1. **Tasks**: Check task status and any task-level errors.
2. **Usage**: Check for credit pressure or unusual spikes.
3. **Billing**: Check plan limits or add-on availability.

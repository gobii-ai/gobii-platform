The Gobii Console is where you manage agents, monitor activity, and control account settings.

## Main navigation

- **[Agents](https://gobii.ai/console/agents/)**: Create agents and open their chat or settings.
- **[Analytics](https://gobii.ai/console/)**: Quick view of activity and trends.
- **[Tasks](https://gobii.ai/console/tasks/)**: Review task status, details, and results.
- **[API Keys](https://gobii.ai/console/api-keys/)**: Create and manage API credentials.
- **[Billing](https://gobii.ai/console/billing/)**: Plan, seats, add-ons, and payment details.
- **[Usage](https://gobii.ai/console/usage/)**: Credit usage and performance over time.
- **[Profile](https://gobii.ai/console/profile/)**: Personal account details.
- **[Advanced (MCP Servers)](https://gobii.ai/console/advanced/mcp-servers/)**: MCP server setup and assignments.

## Personal vs organization context

Gobii supports both personal and organization workspaces. What you see depends on your current context and role.

- In **personal context**, you manage your own agents and billing.
- In **organization context**, access depends on your role (for example, billing or member management permissions).

If a menu item or action is not visible, check:

1. Current context (personal vs organization)
2. Current role permissions in that context
3. Whether that feature is enabled for the account
